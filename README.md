# facebook-live-chat_macro
ก่อนใช้งาน ติดตั้ง package ให้เรียบร้อย\
`npm i eventsource`\
`npm i facebook-sdk`

ก่อนจะใช้งานเช็คให้แน่ใจก่อนว่า ติดตั้ง windows-build-tools ยัง\
`npm install -g windows-build-tools`

`npm i node-gyp`\
`npm i robotjs`
